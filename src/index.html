About YAWA

YAWA (Yet Another Weather App) is a frontend-only web app that fetches real-time weather data from the free Open-Meteo API. Built as a learning project for the Generation MX bootcamp.

  • No API keys required
  • No backend or dependencies
  • No user data stored
  • Input sanitized against XSS
